# Mit Sheth Portfolio

A modern, responsive portfolio website built with React and Vite.

## Features

- 🌙 Dark/Light theme toggle
- 📱 Responsive design
- ⚡ Fast loading with Vite
- 🎨 Beautiful animations and transitions
- 📄 Experience section with company logos
- 🚀 Project showcase
- 📧 Skills display

## Tech Stack

- React 18
- Vite
- CSS-in-JS styling
- Modern JavaScript

## Deployment

This project is ready for deployment on Vercel. Simply connect your GitHub repository and Vercel will automatically build and deploy your portfolio.

### Quick Start

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev` for development
4. Run `npm run build` for production build

## Project Structure

```
src/
├── components/          # React components
├── contexts/           # React contexts (theme)
├── hooks/              # Custom hooks
├── data/               # Static data (experience, projects)
├── styles/             # Global styles
├── App.jsx              # Main App component
└── main.jsx             # Entry point
```
